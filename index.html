Import Guide

一次匯入會長、組長、會員、電話、LINE、組別與豬場資料

建議使用 UTF-8 CSV
本頁用於大量建立會員名錄。可先下載模板，用 Excel 填寫後另存為 CSV UTF-8 再匯入；若用本包附的 Excel 模板，請填寫「會員匯入模板」工作表後另存 CSV。系統會先預覽，不會直接覆蓋資料。

1. 選擇或貼上 CSV

下載範例 CSV 下載 Excel 模板
必要欄位只有「姓名」。身份若空白會自動設為會員；組別若空白會歸為未分組；Token 為舊版個人連結保留欄位，可空白自動產生；一般報名已改用共用連結。電話、LINE、豬場名稱可依隱私需求留白。

2. 欄位格式

欄位說明
姓名必填，會員卡顯示名稱。

電話 / LINE_ID方便組長通知，會員端可依權限隱藏。
會員Token舊版個人連結保留欄位；一般會員報名改用 LINE 群組共用報名連結。

Cloud Sync

Google Sheet 雲端同步

同步設定

將 Apps Script Web App URL 貼在這裡。管理端設定一次後，系統產生的 LINE 報名連結會自動帶入同步設定；會員不用輸入 Apps Script URL，就能送出參加 / 請假資料並回寫 Google Sheet。

尚未同步。

Google Sheet 使用流程

  1. 建立一個 Google Sheet。
  2. 開啟「擴充功能 → Apps Script」。
  3. 貼上本包的 APPS_SCRIPT_CODE.gs。
  4. 設定 ADMIN_TOKEN 與 READ_TOKEN。
  5. 管理端儲存 Web App URL 後，到「LINE 公告 / 分享卡」複製共用 LINE 報名連結。
  6. 會員點 LINE 連結後，系統會自動讀取 Google Sheet 最新課程資料，送出後直接回寫 participantSignup。
  7. 部署為 Web App，執行身分選自己，存取權可選知道連結者。
  8. 把 Web App URL 貼回本頁。
