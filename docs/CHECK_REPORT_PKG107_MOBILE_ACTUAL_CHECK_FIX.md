## 發現問題

管理端「年度會費」頁在手機實測時出現：

```text
ReferenceError: money is not defined
```

原因是年度會費統計卡使用了舊函式名稱 `money()`，但目前系統實際存在的是 `formatMoney()`。

## 修正內容

- 將年度會費頁統計卡改用 `formatMoney()`。
- 補上 `money(value)` 相容函式，避免舊程式碼日後再引用時造成頁面載入失敗。
- 更新主程式為 `app.v107.js` / `app.v107.css`。
- 更新 Service Worker 快取為 `shanxian-pig-study-club-v107`。
