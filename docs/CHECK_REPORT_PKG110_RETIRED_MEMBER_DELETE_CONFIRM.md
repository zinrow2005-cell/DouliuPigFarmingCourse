# PKG110｜退會會員永久刪除提示檢查報告

## 本次修改

- 退會會員名單中的刪除按鈕改成「永久刪除」。
- 從退會會員名單刪除會員時，會先跳出確認提示。
- 確認後會從會員資料庫移除該會員基本資料。
- 同步清除課程中的出席、請假、未回覆、簽到、問卷、課前任務負責人、餐費付款連結。
- 年度會費 / 帳務歷史保留為快照，不讓查帳資料消失。
- 年度會費快照欄位 `memberSnapshotId`、`memberStatusAtRecord` 已納入資料正規化，避免重新載入後遺失退會紀錄標示。

## 實測 / 檢查

- `index.html` 已改為引用 `assets/app.v110.css` 與 `assets/app.v110.js`。
- `sw.js` 快取名稱已更新為 `shanxian-pig-study-club-v110`。
- JavaScript 語法檢查：通過。
- Service Worker 語法檢查：通過。
- Apps Script 語法檢查：通過。

## 使用方式

1. 管理端先把會員設為退會。
2. 進入「退會會員名單」。
3. 點擊該會員的「永久刪除」。
4. 系統會跳出提示，確認後才會清除該會員資料。
